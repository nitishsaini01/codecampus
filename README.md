
# CodeCampus

A student management system for college. Final year college project.

## Tech Stack

- **Frontend:** React
- **Backend:** Node.js
- **Database:** SQL
- **Hosting:** GitHub Pages (`/classnova`)

## Installation

1. Clone the repository
2. Install frontend dependencies: `npm install` (in frontend folder)
3. Install backend dependencies: `npm install` (in backend folder)

## Running

- Frontend: `npm start`
- Backend: `npm run dev`
